Chybí přístup do Notion API

  1. Zkopíruj .env.example do .env.
  2. Vlož NOTION_TOKEN z vlastní Notion integrace.
  3. Nasdílej obě databáze té integraci přes Add connections.
  4. Spusť znovu npm start.
